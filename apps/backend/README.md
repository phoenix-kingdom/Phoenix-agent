# Backend (Node.js + Express)

## Scripts

- dev: `npm run -w apps/backend dev`
- start: `npm run -w apps/backend start`

## Environment

Create a `.env` file in this folder with:

```
PORT=4000
```

## Endpoints

- `GET /health` → `{ status: "ok" }`
- `GET /api/v1` → `{ message: "Backend ready" }`
